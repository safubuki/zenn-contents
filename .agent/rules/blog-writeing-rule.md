# ブログ執筆ワークスペースルール

このワークスペースはブログ執筆のためのものです。以下の構造とルールに従って運用します。

## フォルダ構造

- **articles**
  - ブログ記事の本文（Markdown形式 `.md`）をここに保存します。

- **images**
  - 記事内で使用する画像をここに保存します。

## 画像保存ルール

`images` フォルダ内での画像の管理方法は以下の通りです：

1. `articles` にある記事ファイル名（拡張子 `.md` を除く）と同じ名前のフォルダを `images` 直下に作成してください。
2. その記事で使用する画像ファイルは、作成した同名のフォルダの中に保存してください。

例：
- 記事: `articles/my-blog-post.md`
- 画像フォルダ: `images/my-blog-post/`
- 画像ファイル: `images/my-blog-post/example.png`
