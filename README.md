## Backend - Robot Agent System

This repository also includes a Phase 1 backend infrastructure for an AI-powered robot control system. The backend is located in the `backend/` directory.

### Quick Start

```bash
cd backend
./setup_env.sh    # Linux/macOS
# or
setup_env.bat     # Windows

# Start the server
./start_server.sh # Linux/macOS
# or  
start_server.bat  # Windows
```

For detailed setup instructions, see [backend/README.md](backend/README.md).
